# Barhunter

A web application to discover bars and happy hour deals in your area. Built with React, Vite, and TypeScript.

## Features

- **Map View**: Interactive map to visualize nearby venues.
- **Location Search**: Search for locations to find bars around you.
- **Filters**: Filter venues by various criteria.
- **Venue Cards**: Detailed information about each bar, including happy hour times.
- **Happy Hour Modal**: View specific happy hour details.
- **Notes Modal**: Add personal notes for venues.
- **Custom Venue Modal**: Add custom venues.
- **Local Storage**: Persist your preferences and notes.

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/mngvn/3rdlocationator.git
   cd barhunter
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Start the development server:
   ```
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`.

## Usage

- Enter your location in the search bar to find nearby bars.
- Use the map to explore venues.
- Click on venue cards to view details and happy hour information.
- Apply filters to narrow down your search.
- Add notes or custom venues as needed.

## Technologies Used

- React
- Vite
- TypeScript
- Leaflet (for maps, assuming based on MapView component)
- Overpass API (for data fetching)
